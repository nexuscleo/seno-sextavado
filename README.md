# 🛠️ Calculadora Técnica Nexus

Uma ferramenta web moderna e intuitiva projetada para auxiliar profissionais de usinagem e mecânica de precisão nos cálculos de coordenadas para peças **Sextavadas** e **Quadradas**.

## 🌟 Funcionalidades

- **Cálculo Dinâmico:** Suporte para usinagem interna (medida entre faces) e externa (diâmetro nos vértices).
- **Visualização SVG:** Representação gráfica estática da peça com plotagem automática das coordenadas (X, Y) em cada vértice.
- **Trigonometria Instantânea:** Exibição de valores de Seno e Cosseno baseados na geometria selecionada.
- **Interface Premium:** Design moderno utilizando a técnica de **Glassmorphism**, com temas em tons de azul, verde e vermelho.
- **Totalmente Responsivo:** Adaptado para uso em desktops, tablets e smartphones (chão de fábrica).

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização avançada com Glassmorphism, CSS Grid, Flexbox e Media Queries.
- **JavaScript (ES6+):** Lógica de cálculo trigonométrico e manipulação dinâmica de elementos SVG.
- **SVG:** Renderização de gráficos vetoriais para visualização técnica.

## 📋 Como usar

1. Selecione a **Forma Geométrica** (Sextavado ou Quadrado).
2. Escolha o **Tipo de Usinagem** (Externa ou Interna).
3. Insira o **Diâmetro Nominal** ou a medida entre arestas/faces.
4. Os resultados e as coordenadas nos pontos de interseção serão atualizados automaticamente na tela.

## 📸 Interface

O projeto apresenta um painel de controle à esquerda e um visualizador técnico à direita, garantindo clareza na leitura das coordenadas X e Y necessárias para programação CNC ou traçagem manual.
